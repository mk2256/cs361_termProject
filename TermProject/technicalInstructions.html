Technical Instructions

This web extension utilizes .csv files to import data to Google Calendar. There are a couple of ways to upload. First is to auto-import if no modifications are need. The second way is to download the CSV, where you can then modify the CSV before using the data, either in G-Calendar or another app. To see the documentation, click the link below. Note: if click isn't working, right-click and copy+paste the address into a new window tab.
